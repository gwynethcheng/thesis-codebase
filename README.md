# thesis-codebase

Master's thesis project:

Investigating spatiotemporal trends of phytoplankton in McMurdo Sound, Antartica, using ocean colour products from Sentinel-3 and PACE satellites, as well as the State Space Functional Principal Component Analysis as introduced by Gong et al. 2021 (https://link.springer.com/article/10.1007/s00477-021-02017-w).
